My Work

Applications:

Twitch Stream Viewer: A twitch Viewing app that I created. It allows you to search for a game, and will even predict the game you are typing. It also allows you to view all games, and let's you see top streams (and load more streams). There is a menu that can be opened and closed that allows you to search for specific streamers, with a background color displaying whether they are online or not.

Pomodoro Timer: A pomodoro timer, featuring adjustable session/break time lengths. When the clock hits 0, it will automatically switch to the other.

Javascript Calculator: A calculator I created from vanilla javascript. It has corrections for many invalid inputs.

Full Stack Random Quote Machine: A random quote machine. This machine uses a random quote API that I built myself, using Node.js. Clicking new quote will cause a different quote to appear on the screen, as well as change the color of the screen. Clicking the tweet icon will automatically fill in the quote to a tweet for you.

Websites:

Lafayette Business to Business: A client for Find8 Digital Media. I was heavily involved in the design and creation of the home page.

Right Click Plus: A website created while at Find8 Digital Media. The website was designed and built out by me.

Games:

2048: A version of 2048 that I wrote. Written with vanilla javascript. The "score" is the highest number on the board, and buttons or arrow keys may be used to control the board.

Tic-Tac-Toe: A tic-tac-toe game that I wrote. It features a two player version, where you can face off against someone you're with. It also features a 1 player version against an AI, which was custom written by me.
