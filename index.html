Portfolio

A Twitch Viewer. It uses Twitch's API in order to retrieve data about a specified list of streamers. If they are online, it shows the game they are currently streaming and the current title of their stream.

A weather app using two API's. The first API, ipinfo, tracks the user's location. This data is then given to another API, OpenWeather, which uses the longitude and latitude to show the weather at the user's location. The background image changes according to the weather at the user's location.

A web application that generates random quotes. Users can choose the author that they want a quote by, and a new quote by said person will show up on the quote line. If desired, users can also choose to tweet the quote. A button can be clicked to change the background color if they don't like grey.

A Wikipedia viewer. It takes an input from the user and uses Wikipedia's API to grab the top 5 wikipedia results for that string and a description. The app then displays these results on the page, and users can click it to go to that Wikipedia page.
