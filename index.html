Portfolio

Random Quote Generator

A random quote generator, built using HTML, bootstrap, CSS, and jQuery. Users can click an author they want a quote by, and tweet the quote if they so choose. The background color is changeable as well.

Weather App

My first experiment with API's. It uses the ipinfo.io api to grab the user's latitude and longitude, and uses that information with the openweather api to grab the weather for the location. The info displayed is then changed accordingly using javascript. The background picture changes according to the weather.

Wikipedia Viewer

A wikipedia viewer that I built as a personal project. It takes an input string and then uses it to display the top 5 wikipedia results to the user. I also experimented with more jQuery while doing it. A very fun project!

Twitch Viewer

A Twitch.tv viewer. It shows the statuses of specified streamers. If they are online, it will show the game they are currently playing, and what the title of their stream is. If offline, it will just state they are offline. (Note: if viewed morning-3ish there is a good chance they will all be offline)
